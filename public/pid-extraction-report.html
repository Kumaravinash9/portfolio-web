How this was made. Tags are read from the PDF's own vector text layer (PyMuPDF word boxes) and classified by pattern — equipment E/V/PK-####, valves V*-#####, line numbers size-service-#####-spec, ISA instrument codes, pipe sizes, and notes. Each mark sits at the word's real coordinate — the same word-box grounding the extraction pipeline's annotate stage uses. A production run adds the LLM read for symbols/relationships the text layer alone can't give.
